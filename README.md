# Rural house

## Link

[https://pacog.github.io/maspujou/](https://pacog.github.io/maspujou/)

## Run locally

Install [gulp](https://gulpjs.com/)

Install dependencies:
```bash
npm install
```
Type
```bash
gulp
```

To start a local server and watch the files so it autobuilds.

## Deploy to gh-pages

```bash
gulp deploy
```
